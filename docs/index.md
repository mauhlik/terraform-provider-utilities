# Utility Functions Provider

The Utility Functions provider is used to add utility functions to a Terraform
project. The provider has no configuration.

## Example Usage

**versions.tf**
```terraform
terraform {
  required_version = "~> 1.8"

  required_providers {
    utilities = {
      source  = "mauhlik/utilities"
      version = "1.0.0"
    }
  }
}
```

**providers.tf**
```terraform
provider "utilities" {}
```
